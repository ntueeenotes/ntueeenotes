## Four rules, earned the hard way

```
1. Past papers are non-negotiable. → The single highest-leverage prep tool.
                                     Solve them. Don't just read them.

2. Match prep to platform.        → OASIS rewards question-bank grinding.
                                     Paper exams reward tutorial mastery.

3. Build your own cheatsheet.     → Even when none is allowed.
                                     The act of compiling it is the revision.

4. Start design assignments early. → Especially EE2102, EE3015,
                                     and the Year 4 power electives.
                                     They eat weeks.
```
